JavaScript Language Basics

What is JavaScript

  • Lightwight
  • Cross-platform
  • Object-oriented
- Lightwight: não consome muita memória do computador. - Cross-platform: pode ser usado em múltiplas plataformas e sistemas, não apenas para web dev. - Object-oriented: é basead a em objectos

Primitive Data Types

Type Values
Undefined undefined
Null null
Number -1, 0, 1.5, 5.7e+308, -Infinity, +Infinity, NaN
String 'Ze', "Manel"
Boolean true, false
- A primitive value is an immutable piece of data that is represented directly at the lowest level of the language implementation. Não são objectos como a maioria do resto em js. - Undefined: Data type of a variable that does not have a value yet. So is the value that is automatically assigned if a value is not provided. - Null: 'Non-existent' value that represents the intentional absence of any object value - Number: Floating point numbers, for decimal and integers. Always floating point numbers. All have decimals even we sometimes don't see or even don't define them. - String: Sequence of characters, used for text - Boolean: Logical data type that can only be true or false

Primitive Data Types

JavaScript has dynamic typing: data types are automatically assigned to variables. If later we assign another value with another type, js will automatically change the data type of the variable as well.
